# SpudBot

Welcome to SpudBot, a free, Open Source twitch bot designed by/for TheBlazingSpud. The bot is designed on top of the [TwitchIO python library](https://twitchio.dev) to be used by streamers who are familiar with code, allowing them to edit the code to act as they want. 
It's built for Python 3.12 and makes use of a vairety of public libraries alongside the `spudbot` library.
